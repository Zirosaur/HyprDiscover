# Contributing to HyprDiscover

Thank you for your interest in contributing to HyprDiscover.

## Development Environment

Requirements:

* Fedora Linux
* Python 3.12+
* GTK4
* PyGObject
* PackageKit

Clone the repository:

```bash
git clone git@github.com:Zirosaur/HyprDiscover.git
cd HyprDiscover
```

Run the application:

```bash
python3 hyprdiscover.py
```

---

## Reporting Bugs

When reporting bugs please include:

* Fedora version
* Hyprland version
* Python version
* Error logs
* Steps to reproduce

---

## Feature Requests

Feature requests are welcome.

Before creating a request:

* Check existing issues
* Describe the use case
* Explain why the feature is useful

---

## Pull Requests

Before submitting a pull request:

1. Test your changes.
2. Keep commits focused.
3. Follow existing coding style.
4. Update documentation when needed.

Example:

```bash
git checkout -b feature/my-feature
git commit -m "Add feature"
git push origin feature/my-feature
```

---

## Coding Guidelines

* Use Python 3
* Follow PEP8 where practical
* Prefer readability over clever code
* Keep dependencies minimal
* Avoid unnecessary root operations

---

## Project Principles

HyprDiscover aims to be:

* Fedora-first
* Hyprland-first
* Lightweight
* Secure
* Open source
* Community-driven

---

## Code of Conduct

Be respectful.

Constructive discussion and collaboration are encouraged.

Harassment, discrimination, and abusive behavior will not be tolerated.
