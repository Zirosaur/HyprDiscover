# HyprDiscover Roadmap

## Current Status

Current development stage: MVP

Implemented:

* GTK4 interface
* PackageKit integration
* Update detection
* Update installation
* Update log viewer
* Reboot integration
* KDE Discover integration
* Basic Waybar integration

## v0.2

User Experience Improvements

* Scrollable update logs
* Better error handling
* Disable controls during transactions
* Improved status indicators
* Application icon
* About dialog

---

## v0.3

Package Management Improvements

* Package list view
* Update categories
* Package descriptions
* Update size information

---

## v0.4

Desktop Integration

* Native Waybar integration
* Desktop notifications
* Autostart support
* Background update checks

## v0.5

Flatpak Support

* Detect Flatpak updates
* Install Flatpak updates
* Display Flatpak package list
* Unified update view

---

## v0.6

Native PackageKit Integration

* Replace pkcon subprocess calls
* Use PackageKit D-Bus API
* Improve performance
* Improve reliability

## v0.7

Offline Updates

* Systemd offline updates
* Reboot and install workflow
* Update preparation status
* Offline update monitoring

## v0.8

Polish Release

* Improved UI
* Better accessibility
* More detailed logging
* Stability improvements

## v1.0

Production Release

Goals:

* Stable Fedora Hyprland update manager
* Native PackageKit integration
* Flatpak support
* Offline updates
* Waybar integration
* Community documentation

Long-term vision:

Become the preferred update manager for Fedora Hyprland users.
